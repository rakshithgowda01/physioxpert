# PhysioCare Landing Page - Design Strategy

## Design Philosophy: Modern Medical SaaS

This landing page follows a **premium healthcare SaaS aesthetic** inspired by the Repasse reference design. The design combines professional medical credibility with modern, accessible digital experience.

### Core Design Principles

1. **Trust Through Clarity**: Clean typography, ample whitespace, and organized information hierarchy build confidence in the medical service
2. **Motion with Purpose**: Subtle animations enhance user engagement without distraction
3. **Accessibility First**: High contrast, readable typography, and keyboard navigation throughout
4. **Asymmetric Layouts**: Avoid centered grids; use dynamic left-right compositions for visual interest

### Color System

- **Primary**: #11A7A5 (Teal Medical Blue) - Professional, calming, healthcare-aligned
- **Primary Dark**: #0D7C7A (Darker teal for hover states)
- **Background**: #F6FAFA (Soft off-white)
- **Card Background**: #FFFFFF (Pure white)
- **Text Dark**: #1F2937 (Dark gray for body text)
- **Muted Text**: #6B7280 (Medium gray for secondary text)

### Typography System

- **Font Family**: Inter (modern, clean, professional)
- **Hero Title**: 56px bold (commanding presence)
- **Section Titles**: 36px semi-bold (clear hierarchy)
- **Subtitles**: 20px medium (supporting information)
- **Body Text**: 16px regular (readable, accessible)

### Layout Paradigm

- Asymmetric left-right compositions for hero and features sections
- Floating UI cards for visual interest and depth
- Map-based coverage visualization with pulse animations
- Three-step process flow with connecting lines

### Signature Elements

1. **Floating Cards**: Semi-transparent cards with soft shadows showing patient journey steps
2. **Teal Gradient Sections**: Rich teal backgrounds with subtle gradients for key sections
3. **Pulse Animations**: Glowing location dots on coverage map
4. **Smooth Transitions**: All interactive elements use Framer Motion for fluid motion

### Animation Guidelines

- **Entrance Animations**: Fade-in + slide-up on scroll (staggered for visual flow)
- **Hover Effects**: Lift animation (translateY + shadow increase) on cards
- **Floating Elements**: Gentle up-down motion (2-3px) on hero cards
- **Button Interactions**: Glow effect on hover, smooth color transitions
- **Loading States**: Subtle pulse animations for interactive elements

### Responsive Design

- Mobile-first approach with breakpoints at 640px, 1024px
- Touch-friendly button sizes (min 44px)
- Stack layouts vertically on mobile
- Maintain visual hierarchy across all screen sizes
